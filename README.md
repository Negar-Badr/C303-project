# COMP 303 Project - GROUP 41

## Running Tests
1. Rename the directory:  
   `project` instead of `C303-project` (python doesn't take C303- very well)

2. Then, go in it:  
   `cd project`

3. Use this to run tests:  
   PYTHONPATH="." pytest test -W ignore::DeprecationWarning
